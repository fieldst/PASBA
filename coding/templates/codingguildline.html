E & M Leveling Tool

Per PA&E directive, PASBA will conduct CARA audits using the Army E/M Leveling Tool at all AMEDD sites beginning with the January 2011 audit. The tool is an AMEDD/PASBA developed, modified version of an E/M leveling tool developed by Trailblazer Enterprises, LLC, a Medicare Administrative Carrier. Please refer to the Coding VTC presentation on this subject, April 2010.

DoD Unique Coding Considerations

There are several unique coding considerations that coders and billers face when coding or billing medical encounters in the DoD environment. While many of these issues are unheard of in the civilian community, they are important to the collection of biostatistical data in the investigation of population health issues for DoD.

Occasionally, despite that advancement, some "work arounds" are required to allow us to proceed in our coding while a system change request (SCR) is in development.

Please understand, the AMEDD position is coding accurately to the official guidelines (ICD10-CM, CPT/HCPCS, Coding Clinic, CPT Assistant, and many NCCI and CMS edits).
